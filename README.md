# kash-apps

Standalone web apps, separate from the Pine Script indicators repo.

## Apps

| App | Description | Dev |
| --- | --- | --- |
| [`consistency-tracker/`](./consistency-tracker) | Habit / consistency tracking app (React + Vite) | `npm run dev` |
| [`income-tracker/`](./income-tracker) | German 2026 income & tax calculator (React + Vite) | `npm run dev` |

## Getting started

Each app is self-contained with its own `package.json`. From an app directory:

```bash
cd consistency-tracker   # or income-tracker
npm install
npm run dev
```
